Volet Commentaires

Affiche les commentaires du segment actif s'ils existent dans le fichier source et si le filtre de fichier les prend en charge (ex. : filtre PO).

contient les commentaires ajoutés par l'auteur dans les fichiers PO ou le nom de l'attribut en cours de traduction (pour les fichiers HTML)

Certains formats de fichiers conçus spécialement pour le travail de traduction (ex : les fichiers PO) permettent l'ajout de commentaires. De cette façon, le traducteur peut obtenir le contexte entourant le segment à traduire.

Usage possible : intégrer un avertissement afin que le traducteur vérifie la longueur de la traduction
