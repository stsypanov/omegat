2.1. Nouveau...

Cliquez sur cette option pour créer un nouveau projet.

La création s'effectue à travers deux fenêtres qui s'affichent successivement :

  1. la création du projet vide
  2. le paramétrage du projet et l'ajout des fichiers à traduire

Cette option est grisée lorsqu'un projet est déjà ouvert. Pour créer un nouveau projet dans ce cas :

  • Fermez le projet actuel pour pouvoir en créer un nouveau
  • ou Ouvrez une autre instance d'OmegaT (il est courant d'avoir plusieurs instances d'OmegaT ouvertes simultanément).

La création de projet est une opération relativement rare, qui s'apparente à un paramétrage initial.Projet:Création

Si vous travaillez pour une agence ou un donneur d'ordre, vous n'aurez probablement pas à créer de nouveaux projets, mais plutôt à ouvrir des projets qui auront été préalablement créés et paramétrés.

2.2. Ouvrir...

Cliquez sur cette option pour ouvrir un projet existant.

un nouveau projet.

Cliquez sur cette option si vous êtes invité-e à récupérer un projet OmegaT stocké sur un serveur partagé.

Dans la fenêtre qui s'affiche :

  1. Cochez SVN ou Git selon les instructions qui vous ont été communiquées par l'administrateur du serveur.
  2. Saisissez l'URL qui vous a été communiquée.
  3. Cliquez sur le bouton [...] pour ouvrir votre explorateur et spécifier un nom et un emplacement pour le projet.
  4. Cliquez sur OK.
  5. Saissez le nom et le mot de passe qui vous ont été attribués par l'administrateur du serveur.

Le projet OmegaT est copié sur votre ordinateur.Projet:Copie

Cette connexion au serveur n'est nécessaire qu'une fois. Ensuite, l'accès au projet s'effectue normalement, en local, avec le menu Projet > Ouvrir. Seule différence : OmegaT reconnaît qu'il s'agit d'un projet partagé et synchronise toutes les 2 minutes votre projet avec celui du serveur.

3.2. Volet Dictionnaires

Affiche les entrées de dictionnaires trouvées dans les fichiers dictionnaire situés dans le dossier /dictionary/ si ces entrées sont présentes dans le segment source.

affiche les correspondances du dictionnaire trouvées pour les éléments du segment en cours

Les dictionnaires sont les équivalents électroniques des dictionnaires papier (comme Merriam Webster, Duden, Larousse etc.) que vous pouvez avoir sur votre bureau

3.2.1. Problèmes rencontrés concernant les dictionnaires

Les dictionnaires dans OmegaT utilisent les formats StarDict ou Lingvo DSL. Pour installer les fichiers nécessaires au format StarDict, procédez comme suit : Recherchez la combinaison de langues désirée (ex. : sur la plateforme StarDict ci-dessus) Téléchargez le fichier. Ce devrait être un fichier d'archive tarball (extension tar.bz ou tar.bz2). Utilisez l'utilitaire de désarchivage « untar » (ou l'un de ses équivalents, par exemple winrar pour Windows) pour extraire le contenu du fichier dans le dossier « dictionnary » (dictionnaire) du projet. Trois fichiers comportant les extensions dz, idx et ifo doivent s'y trouver. Veuillez noter qu'en plus des dictionnaires « source-cible », vous pouvez, en utilisant la fonction Dictionnaire, avoir accès à des informations telles que : Webster's Revised Unabridged Dictionary (1913) Longman Dictionary of Contemporary English The Britannica Concise Encyclopedia etc.

3.2.2. Comment télécharger et installer des dictionnaires

Certains dictionnaires sont accessibles sans conditions (ou « libres d'usage ») tandis que d'autres (comme la sélection ci-dessus) sont soumis à la Licence GNU. L'exemple suivant montre le dictionnaire Merriam Webster 10th « en action » : Vérifiez que vos fichiers dictionnaire sont dans le bon dossier (ou à l'intérieur de ce dossier dans un sous-dossier). Vérifiez dans Propriétés → du projet (Ctrl+E). Le dossier contient-il trois fichiers du même nom avec des extensions ? Si un seul fichier est présent, vérifiez son extension. S'il s'agit detar.bz, cela signifie que vous avez oublié de le décompresser (désarchiver).

3.3. Volet Editeur

Pour commencer tout de suite : (texte de l'InstantStartGuide)

volet dans lequel vous tapez et modifiez la traduction

Volet dans lequel vous tapez et modifiez votre traduction. Le volet Éditeur affiche le texte du document partiellement traduit : le texte déjà traduit est affiché dans sa traduction tandis que le texte non traduit reste dans sa langue d'origine. Le texte affiché est séparé en segments. Vous pouvez faire défiler le document et double-cliquer sur n'importe quel segment pour l'ouvrir et le modifier. Dans le cas ci-dessus, les segments déjà traduits sont affichés en jaunes.

Un des segments ci-dessus est le segment en cours. Il s'agit du segment qui est affiché en deux parties. Le partie supérieure est dans la langue source, en caractère gras et sur un fond vert tandis que la partie inférieure correspond à la ligne d'édition et se termine par un marqueur : le marqueur est<segment nnnn> avec nnnn correspondant au numéro du segment dans le projet. Utilisez la partie supérieure comme référence et remplacez ou modifiez son contenu par votre traduction dans la ligne d'édition.

Lorsque vous passez au segment suivant, la traduction est validée et sauvegardée.

Un clic droit sur un segment ouvert (en édition) donne accès aux fonctions suivantes :
